# Production roadmap

## Phase 1 — Current package
- Offline-first 100-day plan
- Syllabus and resources
- 120 starter MCQs
- Quiz, progress and personal current-affairs notes

## Phase 2 — Cloud and editorial platform
- Firebase/Supabase authentication
- Admin dashboard for questions, explanations and current affairs
- Content versioning, source citations and correction reports
- Cross-device sync and encrypted backups

## Phase 3 — Professional learning features
- Daily push reminders and streak recovery
- Adaptive quiz selection based on weak topics
- Full 200-question mock engine and optional negative-marking settings
- Written-exam answer practice, teacher review and viva preparation
- Audio lessons, downloadable notes and bilingual search

## Phase 4 — Commercial release
- Subscription/paywall, coupons and institutional accounts
- App Store/Play Store distribution
- Moderation, privacy policy, analytics consent and accessibility audit
- Licensed large-scale past-paper and model-question bank
