# BCS 100 Pro

A polished, installable mobile-first Progressive Web App (PWA) for completing Bangladesh Civil Service preliminary preparation in 100 days.

## Live app

After GitHub Pages deployment is enabled, the expected project URL is:

`https://cseteesta-bit.github.io/AMR/`

## What is included

- 100-day adaptive study plan: foundation, revision, full mocks and final review
- Official 50th BCS preliminary subject structure and mark distribution
- 10-subject syllabus tracker with topic-level study guidance
- 120 original starter MCQs with answers and explanations
- Timed subject and mixed quizzes (36 seconds per question)
- Study completion, quiz history, accuracy and weak-subject analytics
- Current-affairs note desk with source/date fields
- Verified official and primary learning-resource links
- Dark/light modes, local offline storage and installable PWA support
- Fully responsive mobile UI with no external runtime dependency

## Run locally

1. Install Node.js.
2. Open the project folder in a terminal.
3. Run `npm start`.
4. Open `http://localhost:4173`.

A static web server may also be used. Service workers do not run through the `file://` protocol.

## Install on a phone

Open the HTTPS GitHub Pages URL in Chrome, Edge, or Safari and choose **Install app** or **Add to Home Screen**.

## Content notes

- The syllabus structure is based on the official 50th BCS preliminary syllabus published by the Bangladesh Public Service Commission on 9 December 2025.
- The included MCQs are original starter questions created for practice; they are not presented as official past questions.
- Current affairs change continuously. Verify dates, figures, and policy changes through official or highly reliable sources before memorising them.
- This independent project is not affiliated with BPSC.
